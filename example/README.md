# Vue3 Plotly Demo

Demo of `vue3-plotly` using composition interface.
Implements animation, updating and event click. 
Shows how to import  types from Plotly.js.

### Installation

In `vue-plotly-ts`:
- `npm link`

Then in `vue-plotly-ts/example`:
- `npm install`
- `npm link vue3-plotly-ts`
- `npm run build`

Copy to doc:
- `rm -r ../docs`
- `cp -r dist ../docs`
