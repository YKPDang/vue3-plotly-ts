# Vue3 Plotly Demo

Demo of `vue3-plotly` using composition interface.
Implements animation, updating and event click. 
Shows how to import  types from Plotly.js.

### Installation

In `vue-plotly`:
- `npm link vue3-plotly`

Then in `vue-plotly/example`:
- `npm install`
- `npm link vue3-plotly`
- `npm run build`

Copy to doc:
- `cp -r dist ../../doc`
